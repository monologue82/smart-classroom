## 系统要求

- Node.js 14+
- npm 或 yarn

## 快速开始

```bash
# 克隆仓库
git clone <repository-url>
cd smart-classroom5.1

# 安装依赖
npm install

# 启动服务器
npm start

# 访问应用
打开浏览器访问: http://localhost:3000
```
